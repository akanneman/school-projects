# User Number Analyzer

This is a Java console program that allows a user to enter as many numbers as they want, then displays:

- All numbers entered
- The average
- The smallest number
- The largest number
- The total number of values

How It Works

1. User is prompted to enter a number.
2. After each entry, the program asks if the user wants to keep going.
3. When the user stops, the program:
   - Displays all numbers
   - Calculates and displays stats

Technologies Used

- Java
- ArrayList
- Loops
- Scanner input

 How to Run

1. Clone this repo or copy the `UserNumberAnalyzer.java` file into your own Java environment.
2. Compile and run the file in your IDE or terminal.
3. Enter numbers when prompted, and view the analysis at the end.

Why I Built This

This was a school project to practice Java fundamentals:
- Loops
- Lists
- Input handling
- Simple math operations

Bonus: It makes a clean, readable command-line tool.
