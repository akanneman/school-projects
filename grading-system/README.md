# Grading System

A simple Java console application built for a school assignment.

## What It Does
- Accepts a student's numeric grade (0–100)
- Outputs the corresponding letter grade (A–F)
- Uses conditional logic and console input/output

## How to Run
```bash
javac GradingSystem.java
java GradingSystem
```

## Example
```
Enter grade: 88
Letter grade: B
```

## Technologies
- Java
- Console I/O
- if/else statements
